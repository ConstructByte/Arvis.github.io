Seeds

A Minecraft seed is a unique code that generates a specific world layout, determining the landscape, structures and biomes.
Mods

A Minecraft mod is a user-created add-on that introduces new features, gameplay mechanics, items, or changes to the game's code.
Resource Packs

A Minecraft resource pack is a file that changes the game's assets, including textures, sounds, music, and fonts.
